# 教育分析报告库

> 中小学生作业场景 · 行为与方法训练营  
> 基于真实训练数据的个性化学生分析报告

## 📁 目录结构

```
├── README.md                    # 本文件
├── data/                        # 学生原始数据（JSON索引 + 概览）
│   ├── _index.json              # 全局索引
│   └── _class/                  # 班级维度数据
└── reports/                     # 分析报告
    ├── templates/               # 报告模板
    │   ├── 第一类_试听后转化报告.md     # 转化报告模板
    │   ├── 乔家扬_试听后转化报告.html   # 示例报告HTML源码
    │   └── 乔家扬_试听后转化报告.pdf    # 示例报告PDF
    └── ...                       # 后续报告持续添加
```

## 📊 报告类型

| 类型 | 用途 | 推送时机 |
|------|------|---------|
| ① 试听后转化报告 | 试听3次课后促单 | 试听结束日 |
| ② 正课周报 | 建立持续信任 | 每周一次 |
| ③ 月末续费报告 | 推动续费决策 | 每月末 |
